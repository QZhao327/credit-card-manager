# 💳 Credit Card Optimizer

A web app that helps you maximize cashback by recommending the best credit card for every spending category.

## Features

- Input your monthly spending across 8 categories
- Automatically recommends the best card per category based on cashback rates
- Supports choosable category cards (BofA Customized Cash, Citi Custom Cash)
- Suggests optimal category selection to maximize earnings
- Handles point-based cards (e.g. Hilton Honors) with estimated cash value conversion
- Rotating category warnings with expiry dates (e.g. Chase Freedom Flex)

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS

## Getting Started

Clone the repo and run locally:

git clone https://github.com/QZhao327/credit-card-manager.git
cd credit-card-manager
npm install
npm run dev

Then open http://localhost:3000
